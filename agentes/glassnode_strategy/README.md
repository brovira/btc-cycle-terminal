# Glassnode · Informes estratégicos — material del agente `glassnode_strategy`

Cubre los reportes **largos/periódicos** de Glassnode:
- **Strategy Watch** (mensual) — research.glassnode.com
- **Charting Crypto** (trimestral, con Coinbase Institutional)

Complementa a `glassnode_woc` (Week On-Chain semanal, táctico). Este es **medio plazo / valoración**.

## Estructura
- `reports/*.md` — un archivo por informe, `AAAAMMDD-titulo.md` (fecha = fin del periodo cubierto).

## ⚠️ Copyright
Los reportes de Coinbase/Glassnode **prohíben la reproducción**. Por eso aquí guardamos **RESÚMENES/notas propias** (datos + análisis + evaluación), NO el texto verbatim. El repo es público → nunca pegar el PDF/artículo completo. El PDF original, si se quiere archivar, va al repo PRIVADO (DeFi-Tracker), no aquí.

## Cómo actualizarlo
1. Cuando salga un Strategy Watch (mensual) o Charting Crypto (trimestral), pásame el PDF/enlace.
2. Yo hago el **resumen estructurado** (datos clave + takeaways + una sección "MI EVALUACIÓN") y lo guardo como `reports/AAAAMMDD-titulo.md`.
3. El agente lo lee y responde citándolo.
