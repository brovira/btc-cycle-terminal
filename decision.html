Historial y acierto

El sistema registra cada semana la recomendación y luego mide su resultado (retorno de BTC desde entonces). Aún no hay histórico — se llena solo cada lunes. Para sembrarlo ya: GitHub → Actions → «Decision history snapshot» → Run workflow.
«Acierto»: para acciones de compra/esperar-compra cuenta si BTC subió después; para Tomar beneficio, si bajó después. Con pocas semanas el dato es orientativo — sirve para calibrar los umbrales, no como verdad absoluta.
Cómo se decide (documentación)
Agrega las señales que LMEC y Cowen usan para el largo plazo del ciclo y las traduce a una acción. Precio y BMSB/RSI vía Binance; F&G vía alternative.me; MVRV Z vía Coin Metrics (si el on-chain está disponible). No mide timing intradía ni derivados. Uso educativo — no es asesoramiento financiero.
