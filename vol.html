DVOL — vol implícita (Deribit), ~1 año

Percentil bajo + rango comprimido = muelle cargado. La línea de puntos marca el nivel de hoy.
Cómo leerlo (método vol trader):
  • Régimen = ¿viene movimiento? La compresión (DVOL + BB width bajos, CHOP alto) dice SÍ — pero no la dirección.
  • LP ON/OFF (Choppiness): rango = cobras fees, poco IL; tendencia = riesgo de IL → fuera.
  • VRP = DVOL − RV: negativo → vol barata (comprar straddle); >6 → vol cara (vender premium).
  • Funding / OI / Long-Short / Taker: dónde está cargado el apalancamiento = combustible de squeeze.
  • Dist. 200W MA: tu zona de re-entrada de LP (~$62k). Alerta cuando se acerque.
